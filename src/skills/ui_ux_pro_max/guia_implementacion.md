# UI/UX Pro Max - Design System & Tokens

Reglas de oro para el diseño de interfaces de alta calidad en el proyecto.

## Design Tokens:

- **Glassmorphism**: 
  - Background: `rgba(255, 255, 255, 0.7)`
  - Blur: `blur(15px)`
  - Border: `1px solid rgba(255, 255, 255, 0.4)`
- **Typography**: 
  - Main: 'Inter', system-ui, sans-serif
  - Heading Weight: 800 (Extra Bold)
- **Colors**:
  - Primary: `#646cff`
  - Dark: `#1a1a2e`
  - Success: `#4ade80`
- **Shadows**:
  - Soft: `0 10px 25px -5px rgba(0, 0, 0, 0.05)`
  - Deep (Hover): `0 20px 40px -10px rgba(100, 108, 255, 0.15)`

## Principios UX:
1. **Feedback Inmediato**: Cada acción del usuario debe tener una respuesta visual.
2. **Jerarquía Clara**: El contenido más importante siempre tiene mayor contraste.
3. **Micro-interacciones**: Pequeños movimientos que hacen que la app se sienta "viva".
